Project Title 
Programming of a Windows Sandbox 

Project Description
Dotnetsandbox is a windows desktop application which can be used to run undergraduate student code for the department of computer science at the University of Hull, the application can be started on a windows operating system terminal by simply running the dotnetsandbox.exe executable file located in the bin\Debug folder


*/The single executeable file is located in the .exe file is located in folder bin/Debug*/ 

The Dotnet Zip Folder Contains the following 
-The Zip file of the project  
-User Manual in PDF Format 
-Source code
